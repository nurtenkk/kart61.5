##Toplu Taşıma Kart Uygulaması ##

Kart61.5 , toplu taşıma için geliştirilmiş bir kart yönetim uygulamasıdır.
Kullanıcılar kartlarını yönetebilir, bakiye yükleyebilir ve harcama geçmişlerini görüntüleyebilir.

Backend: Java Spring Boot
Frontend: 
Veritabanı: MySQL

---Özellikler---

-Kullanıcı kaydı ve girişi
-Kart oluşturma ve yönetimi
-Bakiye yükleme ve harcama işlemleri
-Otobüs hatları ve hareket saatleri takibi
-Alarm kurma (hatlar için)

---Kurulum---

-MySQL’de database/schema.sql dosyasını çalıştırın
-Spring Boot uygulamasını başlatın
-Uygulamayı test edin
